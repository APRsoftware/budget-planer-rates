# budget-planer-rates

Statutory tax rate JSON files served as a CDN for [APRsoftware/budget-planer](https://github.com/APRsoftware/budget-planer).

## Files

- poland-{year}.json — Polish statutory rates for the given tax year

## Format

All monetary values in **Grosze** (100 Groszy = 1 PLN). Percentages as plain numbers (e.g. 12 = 12%).

See poland-2026.json for the schema.
